# 时序数据库(Time Series Database)

### InfluxDB

优点：部署方便，单机性能高，Native HTTP API，查询语言强大，搭配telegraf采集数据功能强大
缺点：集群闭源(需付费)

### Graphite

分布式时序列数据存储，容易扩展


### OpenTSDB

容易扩展，集群方便

需要搭建Hbase（存储），ZooKeeper，HDFS，部署复杂

### Prometheus

通过基于HTTP的pull方式采集是序列数据，需要业务端提供http接口

提供整套监控解决方案

### open-faclon

提供整套监控解决方案, grafana 插件版本较低
