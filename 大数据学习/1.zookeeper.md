## 安装配置

分布式协调服务

`export PATH=$PATH:$ZOOKEEPER_HOME/bin`

`cp zoo_sample.cfg zoo.cfg`

`mkdir data log`

`vi zoo.cfg`

```properties
dataDir=/home/user/zookeeper/data
dataLogDir=/home/user/zookeeper/log
```

```shell
zkServer.sh start
zkServer.sh stop
zkServer.sh status

zkCli.sh # 命令行查看
```
